Perguntas e Respostas

  1. Para que serve o atributo "placeholder" na tag <input> com type="text"?

    2. O atributo "placeholder" define um texto que aparece dentro do campo de texto, como uma sugestão para o usuário, até que ele comece a digitar.

  2. Para que serve o atributo "checked" na tag <input> com type="radio" ou type="checkbox"?

    3. O atributo "checked" marca um botão de rádio ou caixa de seleção como pré-selecionado quando a página é carregada.

  3. Para que serve o atributo "value" na tag <input>?

    4. O atributo "value" define o valor inicial do campo de entrada, ou o valor que será enviado quando o formulário for enviado.

  4. Qual atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. O atributo "name" deve ser o mesmo para todos os botões de rádio que pertencem ao mesmo grupo.

  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

    6. A tag <label> é usada para associar um rótulo a um campo de formulário. O atributo "for" vincula o rótulo a um campo de formulário específico pelo seu "id".

  6. Para que serve o atributo "selected" na tag <option>?

    7. O atributo "selected" define qual opção de uma lista suspensa será selecionada por padrão.

  7. Para que serve o atributo "required" nas tags de formulário?

    8. O atributo "required" torna o preenchimento de um campo obrigatório antes que o formulário possa ser enviado.

  8. Para que serve a tag <textarea>?

    9. A tag <textarea> cria uma área de texto multilinha para que o usuário possa inserir grandes quantidades de texto.

  9. Para que serve o atributo "action" na tag <form>?

    10. O atributo "action" define para onde os dados do formulário serão enviados após o envio.

  10. Para que serve o atributo "method" na tag <form>?

    11. O atributo "method" especifica como os dados do formulário serão enviados, podendo ser "GET" ou "POST".

  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

    12. O valor do atributo "method" deve ser "POST" para evitar que os dados do formulário sejam exibidos na barra de endereços.

  12. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

    13. O valor do atributo "method" deve ser "POST".

  13. O que as tags <fieldset> e <legend> produzem?

    14. A tag <fieldset> agrupa elementos de um formulário, e a tag <legend> define um título ou legenda para esse grupo.
